
### 🚀 部署方案

#### 本地开发
```bash
# 一键启动
./deploy.sh

# 或手动启动
python run.py
```

#### 云端部署
```bash
# Vercel部署
vercel

# Docker部署
docker-compose up -d
```

#### 生产环境
- **数据库**: Supabase云数据库
- **服务器**: Vercel无服务器函数
- **CDN**: Vercel全球CDN加速
